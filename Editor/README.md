# Photoportals Tutorial Window

## Overview
The Photoportals Tutorial Window provides quick access to all important resources and documentation for the VRSYS Photoportals package.

## How to Access
In Unity Editor, navigate to:
```
Window > VRSYS > Photoportals Tutorial
```

## Features

### Welcome Section
Provides an overview of the Photoportals package and its purpose based on the research by Kunert et al.

### Quick Start Guide
Direct link to the setup instructions on the GitHub Wiki to help you get started quickly.

### Documentation
Access to the comprehensive package documentation including:
- API Reference
- Usage Examples
- Tutorials

### Research Paper
Link to the original research paper "Photoportals: Shared references in space and time" by Kunert et al. on ResearchGate.

### Example Scene
Quick access to import the example scene through the Unity Package Manager, demonstrating the package capabilities.

### GitHub Repository
Direct link to the source code repository for:
- Exploring the codebase
- Contributing to development
- Staying updated with changes

### Bug Reporting
Easy access to the GitHub Issues page for:
- Reporting bugs
- Requesting features
- Viewing known issues

## Implementation Details

### Files Created
- `Editor/PhotoportalsTutorialWindow.cs` - Main window implementation
- `Editor/vrsys.photoportals.editor.asmdef` - Editor assembly definition

### Menu Location
The window is accessible via: `Window > VRSYS > Photoportals Tutorial`

### Minimum Window Size
400x500 pixels to ensure all content is properly visible.

## Customization
The window uses Unity's EditorStyles for consistency with the Unity Editor theme and automatically adapts to light/dark modes.
