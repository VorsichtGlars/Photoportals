## Color Scheme (Unity Editor Theme Adaptive)

### Light Theme
- Background: Light gray (#C1C1C1)
- Sections: White (#FFFFFF)
- Text: Dark gray (#1A1A1A)
- Buttons: Light blue accent

### Dark Theme
- Background: Dark gray (#383838)
- Sections: Slightly lighter gray (#2B2B2B)
- Text: Light gray (#CCCCCC)
- Buttons: Blue accent

## Typography
- Header: 18pt Bold
- Section Titles: 14pt Bold
- Descriptions: 11pt Regular (Word-wrapped)
- Buttons: 12pt Regular

## Interaction
All buttons open their respective links in:
- Default web browser (for external links)
- Unity Package Manager (for Example Scene)

## Window Properties
- Minimum Size: 400x500 pixels
- Scrollable: Yes
- Resizable: Yes
- Dockable: Yes
